# 水瓶倒水游戏
## 1.游戏介绍
有三个瓶子，容量分别为8升、5升和3升。其中，8升的瓶子装满了水。通过相互倒水，最终获得4升的水。

## 2.操作方式
按照要求输入瓶号，进行倒水操作。

## 3.获胜条件
任意一个瓶子中正好有4升的水。
