# Water Bottle
## 1.游戏介绍
现有三个瓶子,容积分别为8L、5L、3L。<br>
其中容积为8L的瓶子装满水。<br>
试通过来回倒水,最终获得获得4L的水。
## 2.操作方式
按要求输入瓶号,完成倒水操作
## 3.获胜条件
任意瓶中正好是4L水
